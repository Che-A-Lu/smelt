# .card Standard Prompts / .card 标准提示词

> These prompts are the entry point to the `.card` format. Paste them into any AI tool — ChatGPT, Claude, Kimi, DeepSeek — and get structured JSON. Drop that JSON into Smelt Lite to pack a `.card`.
>
> 这些提示词是 `.card` 格式的入口。粘贴到任何 AI 工具中，AI 输出结构化 JSON，拖入 Smelt Lite 即可打包。

**[Open Smelt Lite →](https://che-a-lu.github.io/Smelt-Lite/)**

## Quick start / 快速开始

1. Copy a prompt / 复制提示词
2. Paste into your AI conversation / 粘贴到 AI 对话
3. AI outputs JSON → drag into [Smelt Lite](https://che-a-lu.github.io/Smelt-Lite/) / 拖入 Smelt Lite
4. Download your `.card` / 下载 .card
